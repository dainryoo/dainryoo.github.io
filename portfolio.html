Sapling Academia (2016)

A fake quiz website made for my web development class that stores questions/answers and student scores in a database. Students can take quizzes for different subjects, and teachers can view their results. Sadly, it's not complete, but I learned to use very basic PHP for data retrieval and storage, as well as some D3 for data visualization.
